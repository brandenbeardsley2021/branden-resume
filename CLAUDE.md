# Branden Beardsley — Resume Site

## Project Overview
Static HTML resume site targeting the **Stripe PM, Payments Intelligence** role.
Deployed on Vercel at **branden.cellix.ai**.

## Architecture
- Single `index.html` with inline CSS (no framework, no build step)
- Vercel static deployment via `@vercel/static`
- DNS managed in Cloudflare (A record → 76.76.21.21, DNS only)
